Minesweeper Website

A school project to create a fully interactive Minesweeper game on the web

Overview

This project was created as part of a school assignment to develop a web-based version of the classic game Minesweeper. The game features an interactive grid, multiple difficulty levels, and responsive design for desktop and mobile users.

Case Study

The Minesweeper website project aimed to improve my skills in front-end development and game logic implementation. Key areas of focus included:

  • Dynamic Grid: Generating grids of varying sizes based on difficulty.
  • Game Logic: Implementing the logic for uncovering cells, handling bombs, and calculating adjacent mines.
  • Responsive Design: Ensuring the game works seamlessly across devices.

What I Learned

Through this project, I enhanced my skills in several areas:

  • Mastered DOM manipulation using JavaScript to dynamically update the game board.
  • Learned advanced CSS techniques for responsive design and hover effects.
  • Improved debugging skills while working on game mechanics and edge cases.
  • Understood the importance of user experience (UX) in designing intuitive game interfaces.
