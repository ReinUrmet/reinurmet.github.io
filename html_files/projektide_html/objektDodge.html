Object Dodge Game

A Unity game where players dodge incoming obstacles in a fast-paced environment

Overview

Object Dodge is a Unity game that challenges players to survive as long as possible by dodging incoming objects. The game includes smooth controls, increasing difficulty over time, and a leaderboard to encourage competition. This project demonstrates my ability to work with Unity, C#, and game physics.

Case Study

The Object Dodge project was developed to enhance my Unity game development skills. Here are the key areas of focus during the development process:

  • Player Controls: Implementing smooth and responsive controls for dodging obstacles.
  • Obstacle Generation: Creating a random obstacle spawning system with increasing difficulty.
  • Game Physics: Using Unity's physics engine to create realistic object behavior.
  • Leaderboard: Developing a local leaderboard system to store high scores.

What I Learned

Developing Object Dodge gave me valuable insights into Unity and game development. Key takeaways include:

  • How to use Unity’s physics engine to create realistic and engaging gameplay mechanics.
  • Improved my C# programming skills, especially in implementing efficient scripts.
  • Learned to balance difficulty and gameplay progression to keep players engaged.
  • Gained experience in optimizing game performance for smooth gameplay.
