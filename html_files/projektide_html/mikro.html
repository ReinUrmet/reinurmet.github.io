Mikro Game

A Pygame project where players control Mikro in a world full of challenges

Overview

Mikro is a 2D sidescroller game built using Python and Pygame. In this game, players control a character named Mikro as they navigate through different levels, avoiding obstacles and defeating enemies using unique mechanics. This project demonstrates my understanding of game loops, animations, and event-driven programming.

Case Study

The Mikro project helped me develop skills in game design and implementation. Key areas of focus included:

  • Game Physics: Implementing realistic player movement, gravity, and collision detection.
  • Enemy AI: Designing basic enemy behavior to create engaging gameplay challenges.
  • Graphics and Animations: Creating smooth character animations and a visually appealing environment.
  • Event Handling: Managing user inputs for movement and interaction.

What I Learned

Building Mikro gave me practical experience in game development and programming. Here's what I learned:

  • How to create a game loop to handle updates, rendering, and events efficiently.
  • Enhanced my understanding of Pygame’s features, including surface blitting and sprite handling.
  • Improved problem-solving skills while debugging collision issues and implementing level transitions.
  • Learned how to optimize performance for smoother gameplay on different devices.
